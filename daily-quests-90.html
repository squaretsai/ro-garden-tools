前置任務解法

這裡只放已查到 RO樂園官方公告、RO樂園板攻略或 RO樂園玩家影片可對上的內容。沒有查到樂園專用流程的地區，會標成「待遊戲內確認」。

龜島：前進龜島
樂園玩家影片確認有龜島前置；細節需進遊戲核對
  1. 等級門檻：樂園玩家影片標記為 Lv70 以上；90 等每日角色一定符合。
  2. 到艾爾貝塔上方旅館，進入右側房間，找「烏龜老伯」對話，選擇關於前往龜島的方法。
  3. 到艾爾貝塔右下碼頭找船夫「高丹富羅」約 `244,118`，選擇前往烏龜島。
  4. 支付船費後抵達龜島；之後即可處理龜島相關每日與龜良村任務。

備註：官方 2026/01/13 公告確認樂園開放「亡者國都尼芙菲姆」與「龜島」，但完整前進龜島步驟主要來自樂園玩家影片與既有 NPC 流程，建議第一次跑時按 NPC 對話確認。

崑崙：主線開每日
樂園官方 + 樂園板攻略
  1. Base Lv90 以上，到艾爾貝塔 `246,62` 找「崑崙廣宣大使」，支付 10,000 Zeny 前往崑崙。
  2. 抵達後找「張青龍」`gon_fild01 262,87`，依指示去拜訪「崑崙堂主史艷文」。
  3. 進室內找「崑崙堂主史艷文」`gon_in 17,93`，再依劇情上二樓對話。
  4. 依序跑「散落的道力」與「鐵匠溫伽樂」相關任務。
  5. 回報堂主史艷文。官方公告寫明：完成居民初次的請求後，便能開啟每日任務。
泰國：哎喲泰雅主線 / 神殿入場
樂園官方確認前往方式；神殿流程需遊戲內核對
  1. 先與哎喲泰雅的「福蘭奇斯闊」`ayothaya 157,79` 對話，觸發拜訪哎喲泰雅領袖莫拉4世的劇情。
  2. 找到願意贊助航海費用的贊助者後，再回艾爾貝塔找「航海士安東尼奧」`alberta 246,94` 前往哎喲泰雅。
  3. 若每日要進古代神殿或內部地圖，需跑哎喲泰雅神殿入場線，取得神聖線紗 / 高級神聖線紗類通行道具。
  4. 通用流程會要求準備透明之布、狐狸尾巴、昆蟲硬殼、聖水等材料，並在神殿內尋找 6 個痕跡；樂園版本若 NPC 名稱或材料數量不同，以遊戲內對話為準。
天津：領主的通行證
樂園官方確認 Base90 與前往方式；通行證流程需遊戲內核對
  1. Base Lv90 以上，到艾爾貝塔 `246,34` 找「天津町宣傳使臣」，支付 10,000 Zeny 前往天津町。
  2. 到天津町中間民宅門口，與兩側守門士兵對話，得知城主母親相關事件。
  3. 到天水閣內部四樓找城主「宜希答 佑西那戈」，接下處理城主母親事件的委託。
  4. 回天津町中間找「克武基」對話，再到北方神社找「狐狸面具 / 塔慨虎威」學習解決方式。
  5. 回城主母親住處與老太太互動，成功後取得狐狸尾巴，再回天水閣找城主回報。
  6. 完成後取得「領主的通行證」，即可進天津町密穴 / 天水閣相關地下區域；遺失時可花費 10,000 Zeny 補發。
洛陽 / 龍之城：待確認
目前只確認樂園前往方式
  1. Base Lv90 以上，可在艾爾貝塔 `245,40` 找「龍之城廣宣大使」，支付 10,000 Zeny 前往龍之城洛陽。
  2. 目前未查到可靠的樂園專用「每日必解前置流程」。如果 NPC 不能互動，再依遊戲內提示補跑龍之城本地任務。
死都 / 尼芙菲姆：待確認
官方確認地圖限制，未查到每日前置
  1. 官方公告確認樂園開放「亡者國都尼芙菲姆」，且因地域特殊，無法用傑斯塔、世界移動券前往，也無法 `/memo`。
  2. 目前未查到這組 90 每日需要額外主線前置；重點是先能抵達尼芙菲姆。
