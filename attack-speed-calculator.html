使用公式

可依伺服器差異微調
計算 AGI / DEX = 配點 + 裝備/其他加成 + Job 加成
可用素質點預設依 Level 自動帶入；若與遊戲內不同，可直接修改「可用素質點」欄位。
已消耗點數只計算六圍配點欄；裝備、狀態、Job 加成不消耗點數。
Lv100 前可用素質點：Lv1 起始 48；Lv X 升到 X+1 取得 floor(X / 5) + 3
CRI = 1 + floor(LUK / 3)
攻速上限預設：Base Lv 100 前 190，Base Lv 100 以上 191
ASPD#1 = floor(武器基礎攻速 + 武器/盾牌修正 + sqrt(DEX² / 5 + AGI² / 2) / 4 + ((Status Modifier 總和) × AGI / 200))
ASPD#2 = floor(ASPD#1 + (195 - ASPD#1) × 裝備/技能 ASPD%)
最終 ASPD = min(攻速上限, ASPD#2 + 固定 ASPD+)

左側面板的第二個加號欄位可填裝備、料理、狀態等額外素質；第三個加號欄位請填樂園版 Job 加成。目前沒有抓到可驗證的樂園職業加成表，所以不自動代入原版資料，避免混用版本。攻速公式已依 Google Sheet：覺醒藥水是 Modifier 6，不是 15%；刺客拳刃武器修正為 -2；ASPD#1 與 ASPD#2 都會先無條件捨去。攻速上限只在最後封頂，預設 Lv100 前 190 / Lv100+ 191。可用素質點依 Base Lv 自動帶入，也可手動覆蓋。
